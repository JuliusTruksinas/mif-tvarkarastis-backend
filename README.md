# MIF Tvarkaraštis Back-end

## Projekto pasileidimas

- clone repository
- projekto root folderyje paleisti: **npm install** komandą
- atsisiųsti mongodb serverį į kompą [MongoDb](https://www.mongodb.com/try/download/community)
- pasirinktoje vietoje sukurti tuščią folderį, kuriame bus saugoma mongoDB duomenų bazė, tą folderį pavadinti **"mifTvarkarastisDb"**
- atskirame terminale paleisti komandą: **mongod --dbpath {PASIRINKTA_LOKACIJA}\mifTvarkarastisDb**
- projekto root folderyje paleisti **npm run dev** komandą
